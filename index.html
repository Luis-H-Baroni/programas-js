Linguagem JavaScript

Características

  • Linguagem de Programação
  • Linguagem Interpretada
  • Originalmente criado como uma linguagem front-end, ou seja, ele era interpretado e executado no cliente (Navegador)
  • Também pode ser executado no back-end através do NodeJs
  • É case-sensitive
  • Fracamente tipada, pode ser tipada se programada com "TypeScript"
  • Sintaxe similar com a linguagem C, Java
  • Estruturada, orientada a objetos
